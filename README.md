# OCRGrabber
OCRGrabber is a lightweight desktop OCR tool that lets you extract text from anywhere on your screen using a movable transparent capture window.

## Features
- Transparent, movable capture overlay
- Fast OCR powered by Tesseract
- Supports multiple OCR languages
- Adjustable OCR settings for improved accuracy
- Highlights detected text after each capture
- Auto-capture mode
- Copy or save extracted text
- Multi-monitor support
- Screenshots are processed in memory only and are **not saved to disk**

## How to Use
1. Launch OCRGrabber.
2. Move the capture window over the text you want to read.
3. Click **Capture**.
4. The extracted text will appear in the Results window.
5. Copy or save the text if needed.
